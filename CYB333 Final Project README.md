# Network Reconnaissance Tool
This project is a Python-based Network Reconnaissance Tool that scans a target system for open ports. The tool uses multithreading to improve scanning speed by checking multiple ports simultaneously. It helps security administrators quickly identify active services running on a host.
## Features

 Scan a target IP address for open ports
 Multi-threaded port scanning for faster performance
 User-friendly command-line interface
 Identifies open and closed ports
 Simple and lightweight implementation using Python
